## 🎯 How to Submit (Step-by-Step)

### Step 1: Test Locally (2 minutes)

```bash
cd frontend
npm install
npm run dev
```

Open `http://localhost:5173` and test:
- Type custom text
- Change colors
- Adjust font size
- Try different positions
- Test on mobile view (browser DevTools)

### Step 2: Create GitHub Repository (3 minutes)

1. Go to [github.com](https://github.com) and create a new repository
   - Name: `custom-tshirt-designer` (or your choice)
   - Description: "Responsive custom t-shirt designer with live preview"
   - Public repository

2. Push your code:

```bash
# In the frontend directory
git init
git add .
git commit -m "feat: Custom T-Shirt Designer - Complete implementation

Features:
- Live preview with SVG t-shirt mockup
- Custom text with 6 font styles
- Adjustable font size (16-64px)
- Text positioning (top/center/bottom)
- 6 text colors + 8 t-shirt colors
- Fully responsive design
- Built with React, TypeScript, Tailwind CSS"

git branch -M main
git remote add origin https://github.com/YOUR_USERNAME/custom-tshirt-designer.git
git push -u origin main
```

### Step 3: Deploy to Vercel (2 minutes)

1. Go to [vercel.com](https://vercel.com)
2. Sign in with GitHub
3. Click "Add New Project"
4. Select your `custom-tshirt-designer` repository
5. Click "Deploy" (Vercel auto-detects Vite)
6. Wait ~1 minute for deployment
7. Copy your live URL: `https://custom-tshirt-designer.vercel.app`

The application is production-ready with proper TypeScript types, component structure, and responsive design that works seamlessly across all devices.
